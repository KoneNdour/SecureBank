# SecureBank
Secure Bank est un projet académique réalisé en groupe dans le cadre du module DevSecOps et programmation sécurisée. L’objectif du projet est de concevoir et développer une application bancaire sécurisée intégrant des mécanismes avancés d’authentification, d’autorisation, d’audit, de confidentialité et d’intégrité des données.
